# 运行
运行neo4j （桌面版）, 启动数据库<br>
搭建好环境后,在vue目录下运行: <br>
### npm run dev 或者 cnpm run dev
在 django目录下运行（manage.py 所在文件夹）可能先运行conda环境:
### activate xxx
### python manage.py runserver
